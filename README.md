# Movie-app-clone

> Movie List (Download Count)

## Build Setup

``` bash
# update dependencies
npm update

# serve with hot reload at localhost:3000
npm run start

# build for production with minification
npm run build

# deploy
npm run deploy

# if you deploy error
npm install gh-pages

```
